Desenvolvido como parte da Formação em Lógica de Programação da DIO, este projeto tem como objetivo classificar um herói com base em sua quantidade de experiência (XP). É um exercício de fixação de estruturas básicas da programação, como variáveis, condicionais e operadores.

Objetivo

Exibir o nível de um personagem com base na quantidade de XP, usando estruturas condicionais em JavaScript.

Requisitos Utilizados

  • Variáveis
  • Operadores
  • Condicionais (if / else if / else)

Código do Projeto



// Classificador de Herói
let nome = "Spartacus"
let quantidadeXp = 7634
let nivel = ""

if (quantidadeXp < 1000) {
    nivel = "Ferro"
} else if (quantidadeXp >= 1001 && quantidadeXp <= 2000) {
    nivel = "Bronze"
} else if (quantidadeXp >= 2001 && quantidadeXp <= 5000) {
    nivel = "Prata"
} else if (quantidadeXp >= 5001 && quantidadeXp <= 7000) {
    nivel = "Ouro"
} else if (quantidadeXp >= 7001 && quantidadeXp <= 8000) {
    nivel = "Platina"
} else if (quantidadeXp >= 8001 && quantidadeXp <= 9000) {
    nivel = "Ascendente"
} else if (quantidadeXp >= 9001 && quantidadeXp <= 10000) {
    nivel = "Imortal"
} else {
    nivel = "Radiante"
}

console.log("O Herói de nome " + nome + " está no nível de " + nivel)
